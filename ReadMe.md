# Word Counter

### Requirement:
Build a function that counts how many words exist in a sentence.

### Rules:
words are separated by spaces
multiple spaces may exist
empty input should return 0
## Data Flow
 - raw text-->
 - validated--> 
 - splited to unit of words-->
 - set counter--> 
 - iterated through data--> 
 - count valid words(aggregated)-->
 - print integer count

## Control Flow
- recieve text-->
- validate to check it text is empty--> 
- split string to get unit of words-->
- terate to check for valid words--> 
- aggregate to count the valid words-->
- print output(the integer count)

## Logic Pattern
- the if condition check is to verify if text is empty, 

- the split logic pattern is for modifying or transforming the text to unit of words,

- the loop is for moving through the list of the unit of words and inside the loop, why moving through units the if condition check for valid words and the aggregation logic count the valid words and return the output
